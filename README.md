# 🛠️ Flipkart Clone Backend

This is the backend API for the Flipkart Clone project. It provides RESTful APIs for managing products, cart, wishlist, and orders, enabling seamless communication with the frontend.

---

## 🚀 Features

### 🔗 API Functionalities
- Fetch all products
- Fetch single product by ID
- Add to Cart
- Remove from Cart
- Wishlist management ❤️
- Order processing

### 📦 Product Management
- Product data with pricing, category, brand
- Stock availability tracking
- Image handling (array-based)

### ⚙️ Backend Capabilities
- RESTful API architecture
- Database integration
- JSON-based communication
- Modular route handling

---

## 🛠️ Tech Stack

- Node.js
- Express.js
- PostgreSQL (Database)
- SQL (Queries)
- CORS
- dotenv (Environment variables)
